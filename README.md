# aprendendo GIT

## comandos basicos do git

```sh
git init #para iniciar um repositorio
git remote add origin link_do_repositorio_go_github # pegar o link no github
git add . adiciona todos os arquivos para serem monitorados
git commit -m "com uma mensagem legal entre aspas"
git push irigin main # envia dos dados para o github
git pull origin main # recebe arquivos modificados do github
#mostra se tem arquivos criados, modificados ou excluidos.
# Também informa se o remoto está atualizado com o local e vice-versa
git status
```
